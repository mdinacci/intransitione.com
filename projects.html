iPhone application

An iPhone application I've written for a client that allows the user to build a report in case of an emergency based on pictures, videos, text messages and audio recordings.

The report is then saved on the device and sent to remote servers along with the user position where it will be analyzed and dispatched to the competent authorithies.

Not (yet) available on the iTunes store.
